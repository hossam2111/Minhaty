# مِنَحْتي (Minhaty) — Landing Page

Landing page for Kazakhstan Government Scholarship 2026 — Arabic RTL, mobile-first.

## 🚀 Deploy to Vercel (One Click)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import Project → select this repo
3. Framework Preset: **Other** (plain HTML)
4. Click Deploy ✅

## 📁 Project Structure

```
minhaty/
├── index.html       ← The entire landing page (HTML + CSS + JS)
└── README.md
```

## 🌐 Tech Stack

- Pure HTML5 / CSS3 / Vanilla JS — zero dependencies
- Fonts: [Tajawal](https://fonts.google.com/specimen/Tajawal) + [DM Serif Display](https://fonts.google.com/specimen/DM+Serif+Display) via Google Fonts
- Direction: RTL (Arabic)
- Mobile-first responsive

## 🛠️ Local Development

No build step needed — just open `index.html` in your browser:

```bash
# Option 1: Direct open
open index.html

# Option 2: Local server (recommended)
npx serve .
# or
python3 -m http.server 3000
```

## 🔧 Customization

All editable content is in `index.html`:
- WhatsApp numbers → search `wa.me/`
- Prices → search `price-new` / `price-old`
- Deadline date → search `2026-05-31` in the JS section
- Colors → CSS variables at the top of `<style>` block
